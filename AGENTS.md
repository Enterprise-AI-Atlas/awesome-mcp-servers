# AGENTS.md — Awesome MCP Servers

## Purpose

This repository is a curated registry of Model Context Protocol (MCP) servers, clients, and tooling. It is optimized for agents and coding assistants that need to discover and integrate MCP-capable services.

## Scope

Cover resources that implement or consume the Model Context Protocol:

- MCP servers exposing tools, resources, and prompts.
- MCP clients and SDKs.
- Reference implementations and protocol documentation.
- Community integrations with IDEs, agents, and coding assistants.

## Style

- Follow the same format as the other `awesome-*` repositories in the Enterprise AI Atlas organization.
- Label official/vendor resources as `Official` and third-party resources as `Community`.
- Prefer official documentation and GitHub repository links.

## Maintenance

- Run `./scripts/validate-links.sh` before finishing any edit.
- Categorize entries by the domain or service the MCP server exposes.

## Important reminders

- Do not provide legal advice.
- Verify that links resolve and are current.
